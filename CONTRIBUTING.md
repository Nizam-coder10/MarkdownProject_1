# Contributing Guidelines

Gràcies per voler contribuir a **MarkdownProject_1**!  
Segueix aquestes guies per assegurar consistència i qualitat.

## Estil de Codi
- Utilitza indentació clara i coherent.
- Escriu missatges de commit descriptius i en anglès.
- Evita codi duplicat i segueix bones pràctiques.

## Pull Requests
- Crea una branca per cada funcionalitat o bugfix.
- Assegura’t que el codi compila i passa les proves abans de fer la PR.
- Escriu una descripció clara del que fa la PR.

## Report d'Issues
- Utilitza la pestanya **Issues** de GitHub.
- Dona un títol clar i una descripció detallada.
- Inclou passos per reproduir el problema si és possible.
