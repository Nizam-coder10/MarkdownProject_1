## Introduction

Per més informació, pots mirar la [Documentació de Git](https://git-scm.com/docs/git).

## Features
Aquest projecte posa en pràctica diversos conceptes bàsics de Git i Markdown que faciliten el treball amb repositoris i documentació tècnica. A continuació es mostren les funcionalitats implementades fins ara:

| Feature             | Description                                                                 | Status        |
|---------------------|-----------------------------------------------------------------------------|---------------|
| Git Init & Commit   | Inicialització del repositori i ús de commits amb missatges clars           | Done          |
| Markdown Structure  | Creació d’un README amb encapçalaments, llistes, taules i imatges            | In Progress   |
| .gitignore Setup     | Configuració del fitxer .gitignore` per ignorar contingut específic        | Not Started   |
